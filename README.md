# Aplicación inteligente para la detección de plagas agrícolas

Este proyecto corresponde al desarrollo de una aplicación móvil inteligente para la detección de plagas agrícolas mediante clasificación de imágenes utilizando aprendizaje profundo.

El sistema emplea redes neuronales convolucionales (CNN) para analizar imágenes de cultivos y determinar el tipo de plaga presente, funcionando de manera local y sin conexión a Internet.

## Objetivo del proyecto

Desarrollar un modelo de inteligencia artificial capaz de identificar plagas agrícolas a partir de imágenes, el cual será integrado en una aplicación móvil Android orientada a pequeños y medianos agricultores.

## Tecnologías utilizadas

- Python 3.9
- TensorFlow y Keras
- OpenCV
- NumPy
- Android (Kotlin)
- SQLite
